## Image Optimization

For the best performance on GitHub Pages:
- Convert all images to WebP format when possible
- Keep file sizes under 400KB
- Use appropriate dimensions (suggested: 1200px width max)
- Add "loading=lazy" to all img tags
- Optimize with tools like TinyPNG or Squoosh
